BRANCH & MERGE

Learn Git Branching

Git Commands 


			
			# list your branches. a * will appear next to the currently branch
			git branch 

			# create a new feature branch at the current commit	
			git branch feature-1 

			# create a new feature branch at the current commit & checkout		
			git checkout -b feature-1 

			# switch to another branch and check it out into your directory		 
			git checkout feature-1

			# switch to another branch and check it out into your directory
			git switch feature-1 
			
			# To delete the branch
			git branch -d my-branch-name
			
			# To Delete the branch with Force
			git branch -D my-branch-name

			# To Merge the feature branch to main branch (master)
			git merge [branch]
